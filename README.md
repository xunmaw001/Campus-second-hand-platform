# Campus-second-hand-platform
这款校园二手微信小程序，为师生打造便捷交易平台。在这里，你能一键发布闲置物品，像教材、电子产品、生活用品等，附上清晰图片与详细描述，快速吸引买家。还可精准搜索心仪好物，按品类、价格筛选。内置聊天功能，方便买卖双方沟通细节。支持线上协商定价、线下当面交易，安全又省心。交易完成后能互评，积累信誉，为校园资源循环利用助力 。
